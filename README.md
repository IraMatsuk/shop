# Online Shop "Card4You"

### Guest can:
- change language
- find all postcards in the catalog
- register
- sign in, if he is registered
- sort all postcards by price
### Client can:
- change language
- sign in
- sign out
- update profile
- change password

### Admin can:
- change language
- sign in
- sign out
- update profile
- change password
- block/unblock clients
- delete admins
- manage sections
- add new admins
